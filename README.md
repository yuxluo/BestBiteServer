# BestBite Server

Yo yo! 歡迎來訪BestBite服務器端的Github頁面！BestBite是一個由BestBite責任有限公司開發的吃貨軟件，請訪問BestBiteClient來獲取更多資訊。

## Getting Started

Clone到Apache/Nginx裏，然後跑server.py和parser.py

server.py會更新Menu.txt和LatestVersion.txt，parser.txt會生成Today.txt

## Prerequisites

必須支持wget和python3
